Hello! Welcome to my portfolio page. Here are my projects from my first year study of front-end development at noroff.

GameHub - Online shop

GameHub is an fictional shop where users can purchase computer games online. The online shop have an add to cart and save game feature which is stored in the local storage. The products are fetched from a WordPress API and the details page is dynamically built showing the game that gets clicked. The website is responsive across screen sizes.

Community Science Museum

The museum wanted a website that is informative, appealing and that will attract both middle school pupils and their parents. The website is informative and engaging, but also encourage viewers to visit the museum itself. This website is responsive, following WCAG principles and DRY principle. The website also contains 4 different pages and are built with HTML and CSS.

Repository Live site

Goldy - Blog

This blog is about the dog Goldy and contains stories about her daily adventures. The website is built with HTML, CSS and JavaScript while fetching content from WordPress REST API. The site is responsive and easy to use on a variety of devices and screen sizes. The blog follows the WCAG principles and DRY principle. This website contains 5 different pages.
